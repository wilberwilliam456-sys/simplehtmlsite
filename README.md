# The Beauty of the Ocean

A simple, single-page website that showcases the beauty of the ocean through an image and descriptive text.

## Overview

This project is a static HTML page (`oceansite.html`) built with pure HTML and CSS — no external libraries or frameworks. It features a clean, classic design with a serif font and warm earthy tones.

## Page Structure

- **Header** — Displays the page title on a dark teal background.
- **Image Section** — Shows a beach photograph (`beach.jpg`) with an italic caption.
- **Content** — A short paragraph reflecting on the calm of the ocean.
- **Footer** — A simple copyright notice.

## Files

| File             | Description                        |
| ---------------- | ---------------------------------- |
| `oceansite.html` | Main HTML page with embedded CSS   |
| `beach.jpg`      | Beach photograph used on the page  |

## How to View

Open `oceansite.html` in any modern web browser — no server or build step required.

## Technologies

- HTML5
- CSS (embedded in `<style>` tag)
